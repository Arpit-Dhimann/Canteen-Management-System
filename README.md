# Canteen Management System

A console-based application designed to simulate a real-world canteen order system using C programming and data structures. This project was developed during my 1st year of engineering to apply theoretical knowledge of data structures into a functional system.

---

## 🚀 Features

- **Categorized Menu**: Display items under Breakfast, Main Course, Snacks, and Beverages.
- **Order System**: Place orders with automatic bill calculation.
- **Queue Implementation**: 
  - **Circular Queue** for student orders.
  - **Priority Queue** for staff orders.
- **Order Management**: Cancel orders using Order ID and track order history.

## 🛠 Concepts Used

- **Structures in C**: Custom data types to manage menu items and order details.
- **Data Structures**: Implementation of Circular Queue and Priority Queue.
- **Modular Programming**: Use of functions for clean logic separation.

## 📅 Project Background

This project was developed during my **1st Semester** of engineering. It was my first major attempt at building a console-based application to apply theoretical knowledge of C Programming and Data Structures into a functional system. Building this helped me understand how to manage data efficiently using Queues and Structures, and it laid the foundation for my interest in Backend and System Design.
